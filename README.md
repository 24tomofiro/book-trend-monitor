# Book Trend Monitor 📚

特定の書籍に関する世の中の盛り上がりを、Web全体とX（旧Twitter）から自動で収集・可視化・分析する自動化ツールです。

## 🌟 主な機能

* **自動情報収集**: Google Custom Search APIを利用し、Web全体とX（`site:x.com`）での言及数を取得。
* **定期実行（1日4回）**: GitHub Actionsにより、日本時間の朝・昼・晩・深夜のタイミングで完全自動で稼働。
* **次世代インタラクティブ・レポート**: Plotlyを採用したブラウザ操作可能なHTMLレポートを生成。
    * **左右2軸グラフ**: 言及数の異なる「X投稿数（左軸）」と「Web全体（右軸）」を最適に比較。
    * **クリック連動型リンク表示**: グラフ上の点を選択すると、その瞬間の注目リンクをグラフ下部に即座に表示。
    * **レンジスライダー**: 期間選択バーにより、蓄積された大量のデータから特定の期間を直感的に分析。
* **感情分析（ポジネガ可視化）**: 投稿内容の簡易スコアリング（0.0〜1.0）を行い、評判の推移をエリアチャートで表示。
* **柔軟な設定管理**: 調査対象や除外キーワードは `config/books.yaml` で一括管理。
